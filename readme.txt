== Description ==

It seems that Search Engine visitors are more likely to click on targeted advertisements then your regular readers. With this plugin you can choose if you want or not show AdSense only to Search Engine Visitors. and social networking, such as Facebook, Twitter and etc.


== Installation ==

1. Upload the zipfile to the `/wp-content/plugins/` directory and unzip
2. Activate the plugin through the "Plugins" menu in WordPress
3. Configure the plugin
